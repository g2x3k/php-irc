PHP-IRC(this is running on php-irc http server)

Introduction

PHP-IRC is a modular IRC bot written in PHP5 with a combination of object based/oriented programming. It operates under several different platforms with the purpose of automating common information related tasks on IRC. There are two main reasons that PHP was selected as the primary language. First off, I wanted to provide novice programmers a way to code their own functions and abilities into a powerful platform without having to worry about such things as memory management and compilers. Secondly, I wanted to do something that no one has done before--create a fully featured bot in php. I don't know if I will ever complete the second objective, but I will keep on developing, as it will always be the main goal of this project. Nevermind the fact that I get constantly ridiculed by all my friends for doing this... hehe :)

Current Major Features

  • DCC Chat Support with custom DCC Chat handlers via user-defined modules
  • DCC File transfer support with NAT and mIRC DCC Reverse protocol support
  • Modular based function support
  • Event based, with user-definable event handling in a simple to use configuration file
  • Multiple bots under one process
  • Remote administration via DCC Chat interface or private message
  • MySQL support, and a custom *.ini engine for users with no access to database
  • Fully documented features and command reference, including in-depth PHP5 installation instructions
  • Just to name a few, and more is in the works!
